# AI-Based Crop Disease Detection System Using Images

## Project Overview

The AI-Based Crop Disease Detection System is a machine learning and computer vision application designed to identify crop diseases from images of plant leaves. The system helps farmers and agricultural experts detect diseases at an early stage, enabling timely treatment and reducing crop losses.

Using deep learning techniques, the model analyzes leaf images and classifies them into healthy or diseased categories. The system provides accurate predictions along with disease information and recommended treatment measures.

## System Workflow

1. Capture or upload a crop leaf image.
2. Preprocess the image.
3. Extract important features.
4. Pass the image through the trained CNN model.
5. Predict the disease class.
6. Display disease details and treatment suggestions.

---

## Machine Learning Model

The system uses a Convolutional Neural Network (CNN) trained on a dataset of healthy and diseased crop leaf images.

### Model Tasks
- Image Classification
- Feature Extraction
- Disease Identification
- Prediction Generation

## Installation

Clone the repository:

```bash
git clone https://github.com/pratikshagujale5-gif/AI-based-crop-disease-detection-system-using-images.git
```

Move to the project directory:

```bash
cd AI-based-crop-disease-detection-system-using-images
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
